# Athlete Injury Risk Assessment System (JUDO)

An AI and Machine Learning based sports analytics project focused on predicting injury risk in athletes using training workload, fatigue levels, competition exposure, recovery habits, and weight-cutting patterns.

## Project Overview

This project aims to help athletes and coaches monitor injury risk through data-driven analysis and predictive modeling.

The system analyzes factors such as:

- Training intensity
- Randori duration
- Grip fatigue
- Lower body fatigue
- Weight cutting
- Competition frequency
- Recovery habits
- Coach supervision

Using Machine Learning, the system predicts whether an athlete is at high risk of injury and provides personalized recommendations.

---

## Features

- Real-world athlete survey dataset
- Synthetic athlete data generation
- Exploratory Data Analysis (EDA)
- Injury risk prediction using ML models
- K-Fold Cross Validation
- Feature importance analysis
- Streamlit web application
- Personalized injury prevention recommendations

## Machine Learning Models

The following models were explored:

- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting
- Support Vector Machine (SVM)

Random Forest was selected as the best-performing model for injury risk prediction.

---

## Dataset

The dataset contains athlete-related information including:

- Physical attributes
- Training workload
- Fatigue levels
- Recovery practices
- Competition participation
- Injury history

Both real-world survey data and synthetic data were used for experimentation and model training.
